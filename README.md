# BlackShield Transportation Website

Phase 1 static website for a luxury chauffeur, black car, sprinter, and optional armed security chauffeur service.

## Files

- `index.html` — homepage
- `style.css` — website styling
- `script.js` — mobile menu and vehicle selection behavior

## Deployment

Upload these files to GitHub, then deploy the repository to DigitalOcean App Platform as a Static Site.
